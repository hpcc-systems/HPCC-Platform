# Contributing to HPCC-Platform

Thank you for your interest in contributing to the HPCC Systems Platform!

We welcome contributions from the community. Please read our detailed [contributor guidelines](CONTRIBUTORS) for information on:

- Setting up your development environment
- Creating topic branches and pull requests
- Commit message guidelines
- Code standards and review process
- Testing requirements

## Quick Links

- **Issues**: [HPCC Systems JIRA](https://hpccsystems.atlassian.net/secure/Dashboard.jspa)
- **Documentation**: [Developer Documentation](https://hpcc-systems.github.io/HPCC-Platform/)
- **Pull Request Template**: [.github/PULL_REQUEST_TEMPLATE.md](.github/PULL_REQUEST_TEMPLATE.md)

## Important Notes

- All pull requests must reference an open JIRA issue (format: `HPCC-nnnnn`)
- All commits must be signed
- Code must follow our coding standards
- New features should include appropriate tests

For complete details, please see the [CONTRIBUTORS](CONTRIBUTORS) file.
