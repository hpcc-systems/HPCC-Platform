HPCC : 9.10 Platform Release Announcement ​

We're pleased to announce the next generation of the HPCC Systems® Platform. This exciting new release, version 9.10.0, provides many enhancements to lower cloud costs, improve performance, and provide a better user experience. Additionally, the latest release now includes all plug-ins, eliminating the need for any separate plug-in installations.

Highlights ​

  • Cloud Cost savings

  • Performance (Thor)

  • Performance (Roxie)

  • Logging

  • Efficient File Copy on Cloud Deployments
    Traditional DFU Server file transfers involve reading files block-by-block from the source and writing them block-by-block to the destination. This approach not only ties up cluster resources but also incurs costs for each block read and written, as well as for ingress and egress bandwidth.

    By offloading the copying of files hosted on storage planes to cloud providers, we can achieve greater efficiency. Major cloud providers offer storage services with APIs that facilitate file transfers between storage locations. Some APIs even support transfers between different cloud providers. For instance, Azure's API allows data transfers between Azure storage and Amazon S3 buckets.

    These API-driven file transfers are typically more efficient and cost-effective than traditional DFU Server transfers. HPCC Systems now supports API copy for Azure Blob Storage and Azure File Shares.
