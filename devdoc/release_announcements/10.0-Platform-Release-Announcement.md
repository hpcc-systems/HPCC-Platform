# HPCC : 10.0 Platform Release Announcement

We're excited to announce the latest release of the HPCC Platform, packed with significant enhancements that streamline your data analytics workflow and boost productivity. This release focuses on reducing cloud costs, improving user experience, strengthening security, and expanding ECL language capabilities.

## What's New in 10.0

### Enhanced ECL Watch Interface

- **New Activities Page**: Navigate and monitor your platform activities with our redesigned activities dashboard, providing better visibility into system operations and job status. *(Tech Preview)*

### XREF

- XREF now supports containerized deployments.

### Strengthened Security & Authentication

- **Improved LDAP Integration**: Enhanced LDAP authentication mechanisms with improved error handling and connection stability.
- **Security Fixes**: Multiple security improvements to ensure your data remains protected and compliant with enterprise standards.

### ECL Language Enhancements

- **New `REGEXEXTRACT` Keyword**: Powerful new ECL function for advanced pattern matching and text extraction, enabling more sophisticated data parsing capabilities.
  For more details, see: [REGEXEXTRACT Docs](https://hpccsystems.com/wp-content/uploads/_documents/ECLR_EN_US/REGEXEXTRACT.html)

- **Modernized Cryptography**: Replaced the legacy `Std.Crypto` library with the robust `Std.OpenSSL` library, providing better performance, enhanced security algorithms, and improved maintainability.
  For more details, see: [OpenSSL Migration Guide](https://hpcc-systems.github.io/HPCC-Platform/devdoc/userdoc/Migrating-from-StdCrypto-to-StdOpenSSL.html)

### Performance & Reliability Improvements

- Numerous cost reduction and cost tracking enhancements.
- Enhanced system stability across Thor, Roxie, and Dali components.
- Optimized memory management and resource utilization.
- Improved error handling and logging throughout the platform.

## Get Started Today

Ready to experience these exciting new features? Download the latest release and explore how these enhancements can accelerate your data analytics projects. We encourage you to try the new ECL Watch activities page and experiment with the `REGEXEXTRACT` functionality in your ECL code.
