
# How To... Videos - HPCC Systems Cloud Native Platform

The following videos are specifically related to getting started using our Cloud Native platform.

---

## How to&#x2026;

- [Deploy an HPCC Systems instance on the Azure cloud using Terraform modules](https://youtu.be/Ck_FV0oGkbM)  
  Manually deploying HPCC Systems® to a cloud platform repeatedly can become tedious, inconsistent, and potentially introduce errors over time. This video demonstrates how to use an "Infrastructure-as-code" orchestration tool—Terraform and the modules we provide to consistently deploy your HPCC Systems to the cloud.
- [Containerized HPCC Systems Platform - Job Cost Tracking](https://www.youtube.com/watch?v=-oObqLhYQks)  
  Introduces new features, shows some of the configurable settings, and touches on how to customize these settings.

- [Start Kubernetes Dashboard on Docker Desktop Kubernetes](https://youtu.be/EfCxTiRxnBw)
- [Setup a default HPCC Systems Deployment on Docker Desktop](https://youtu.be/FRpOSg088v8)

  Learn how to install and run the HPCC Systems platform in a Docker Desktop containerized environment on a Windows PC.

  Setting up and installing local persistent storage volumes for a containerized HPCC Systems deployment. Specifically, on a Windows PC platform employing Docker Desktop containers utilizing Hyper-V as the backend.
